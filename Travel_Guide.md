# 01 准备阶段

> [!info] 💡 核心原则
> 7月的法国正值盛夏，南部（马赛、圣保罗三堡）阳光热烈，北部（巴黎、圣马洛）早晚可能有温差。精简行李，留出空间，随时准备迎接高强度的徒步与城市暴走。

## 📱 1. 数字生活 (必备 App 下载与注册)

### 交通导航
- [x] **Google Maps**: 全局导航，出国前记得提前下载好沿途几个城市的离线地图。
- [x] **SNCF Connect**: 法国国家铁路官方 App，用于购买和查看 TGV、TER 火车票以及扫码进站。
- [x] **Bonjour RATP**: 巴黎大区公共交通官方 App，查地铁线路、公交极其精准。
- [x] **Uber / Bolt**: 打车软件双保险（Bolt 在欧洲有时候比 Uber 便宜）。

### 住宿与财务
- [x] **Booking.com**: 随时查看已预订好的酒店订单、获取门禁密码或联系房东。
- [x] **中国银行 App**: 随时随地查汇率，零额度 VISA 卡余额不足时可即时购汇美元补充。

### 语言与网络通讯
- [x] **Google Translate / DeepL**: 提前下载法语离线语言包，应对个别菜单或突发交流。
- [x] **TheFork**: 欧洲版“大众点评”，用于提前预订法餐厅（部分时段有折扣）。
- [ ] **网络方案 (必须落地即通)**: 
  - **首选**: 手机支持的话，提前在 **Airalo App** 购买法国 eSIM (约 $15/10GB)。
  - **备选**: 落地后在 CDG 机场购买 **Orange Holiday** 实体卡 (约 €40，含大量流量与通话)。

### 核心物资

- [x] **护照原件** (含已签发的申根签证)。
- [ ] **重要文件复印件**：护照首页、签证页复印件（与原件分开存放，防万一）。
- [ ] **纸质版预订单**：将所有 Booking 酒店预订单、火车票打印一份纸质版（过海关和前台 Check-in 时非常高效）。
- [x] **实体卡片**：中国银行 VISA 卡（主刷）、银联借记卡（备用取现）。
- [ ] **欧元现金**：国内提前兑换好的 250-300 欧元零钱（10欧、20欧面额，**切记预留出 200 欧作为圣马洛公寓押金**）。
- [x] **学生证**：带上清华的实体学生证或办理一张 ISIC 国际学生证，欧洲许多博物馆和景点对 25 岁以下学生有隐藏折扣或免票政策。

### 🚨 补充：巴黎交通票务线上 Bug 预案

> [!danger] 线上防卡死提示
> 如果由于 Cloudflare 网络风控导致无法线上注册账号，或由于安卓系统 NFC 兼容性导致 App 报错提示“版本不够新”，**立刻放弃线上折腾**，转为落地后线下办理实体卡。

- [ ] **准备 1 寸免冠证件照（2-3张）**：
  - **尺寸要求**：约 25mm × 30mm（国内普通的 1 寸白底或蓝底照片即可）。
  - **用途**：落地后办理实体的 **Navigo Découverte**（面向游客的周卡实体卡底卡）。法国查票极严，实体周卡上**必须**贴照片并写上自己的名字，否则被查到会面临高额罚款（甚至可能高达 50 欧）。
- [ ] **落地线下办理流程**：
  - **地点**：7 月 2 日清晨 05:50 落地 CDG 机场后，直接步行至 Terminal 2 的 **[SNCF/RATP 联合售票中心 (Gare RER)](https://www.google.com/maps/search/?api=1&query=Gare+RER+Aéroport+Charles+de+Gaulle+2+TGV)**。
  - **话术**：直接对柜台工作人员说：*"Bonjour! Je voudrais une carte Navigo Découverte avec un forfait semaine pour la zone 1 à 5, s'il vous plaît."*（您好，我想办一张 1-5 区的 Navigo Découverte 周卡）。
  - **费用**：5 欧元工本费 + 约 30 欧元左右的当周周卡充值费（直接刷你的中行 VISA 美元卡，选择 EUR 结算）。
  - **后续**：拿到实体卡后，立刻把准备好的证件照贴在上面，并在名字栏写上 `Song Hu`。这张卡可以一直用到 7 月 5 日周日晚上，无限次刷闸机进站。

## 🔌 3. 电子设备 (记录与续航)

- [ ] **电源转换头**：欧洲标准（欧标，两个圆柱形插脚），建议带 2 个。
- [ ] **大容量充电宝**：旅行在外导航、拍照极其耗电，建议带 10000mAh - 20000mAh，支持快充。
- [ ] **主摄影机 (数码相机)**：+ 备用电池 + 大容量 SD 卡 + 读卡器。
- [ ] **DV 摄像机**：+ 对应的充电线材与备用带/内存卡（用来记录沿途动态影像极佳）。
- [ ] **多口 GaN (氮化镓) 快充头**：一个头解决手机、耳机、充电宝的充电需求。
- [ ] **各类数据线**：Type-C、Lightning 备齐。

## 💊 4. 药品与健康 (有备无患)

> [!warning] 注意
> 法国药妆店（带绿色十字标志）随处可见，可以买到很好的护肤品，但处方药购买极严，感冒消炎等常备药必须自己带足。

- [ ] **感冒与退烧药**：布洛芬（Ibuprofen）、对乙酰氨基酚（泰诺/散列通）等。
- [ ] **肠胃药**：蒙脱石散、健胃消食片、黄连素（应对水土不服或饮食油腻）。
- [ ] **外伤急救**：创可贴、碘伏棉签。
- [ ] **防磨脚贴/水泡贴**：**极度重要！** 尤其是应对卡朗格峡湾和枫丹白露的高强度徒步。
- [ ] **晕车/晕船药**：乘坐 TGV 跨城或前往[迪纳尔](https://www.google.com/maps/search/?api=1&query=Dinard)（Dinard）乘渡轮时可能会用到。
- [ ] **防晒高阶装备**：SPF 50+ 的高倍数防晒霜、晒后修复芦荟胶（南法阳光毒辣，地中海沿岸极易晒伤）。

- [ ] **夏季日常休闲装**：T恤、衬衫、休闲长裤
- [ ] **墨镜**：防晒且出片。
- [ ] **遮阳帽 / 运动帽**。
- [ ] **轻便双肩包**：作为 Daypack，装水、相机、随身外套和路餐。

## 🪥 6. 洗漱与日用 (欧洲环保特色)

> [!info] 酒店提示
> 欧洲绝大多数酒店（无论是青旅还是高档公寓）出于环保，**绝对不提供**牙刷和牙膏。拖鞋也仅有少部分提供。

- [ ] **牙刷、牙膏**。
- [ ] **旅行装洗护用品**：部分民宿公寓可能洗发水沐浴露储备不足，带一套旅行分装瓶以防万一。
- [ ] **轻便拖鞋 / 人字拖**：洗澡和在公寓内放松时使用。
- [ ] **旅行便携雨伞**：虽然是夏季，但巴黎等地偶尔会有阵雨。
- [ ] **个人护肤品**、剃须刀。
- [ ] **便携纸巾 / 湿厕纸**。

# 03 每日探索路线

## 3.1 巴黎 (Paris) | 7.2 - 7.7
### 🛬 抵达与交通
05:50 抵达[巴黎戴高乐机场 (CDG)](https://www.google.com/maps/search/?api=1&query=Aéroport+Charles+de+Gaulle+Paris)。

- **办理 Navigo 周卡 (Navigo Semaine)**：前往机场 RER 站服务台购买并充值。需自备 1 寸免冠证件照，购卡后务必贴上照片并签名，以符合实名制查票要求。
    
- **前往市区**：自 2E 航站楼提取行李后，循 "Paris par Train (RER B)" 指示牌步行至 2 号航站楼自带的高铁与快线站 (Aéroport Charles de Gaulle 2 TGV)。刷 Navigo 卡进站，乘坐开往巴黎市区方向 (Paris / Robinson / Saint-Rémy-lès-Chevreuse) 的 RER B 线。
    
- **[巴黎北站 (Gare du Nord)](https://www.google.com/maps/search/?api=1&query=Gare+du+Nord+Paris) 下车**：车程约 35-40 分钟。到站后下车，出站步行 1.1km 前往住宿地。

# 04 独行护航：安全与生存法则

> [!tip] 💡 独行者的小提醒
> 您一个人旅行，保持警惕与节奏是核心。法国虽然浪漫，但对陌生环境的“社会经验”同样重要。

### 🚨 1. 独行防盗指南
- **地铁防抢**: 在巴黎地铁(特别是 1、4、B线)，**绝对不要在车门即将关闭时玩手机**。抢匪通常会守在门边，在关门瞬间抢走手机并冲向站台。
- **TGV 行李安全**: 在高铁大件行李架放行李时，尽量放在视线可见范围内。列车在中途站停靠时，建议起身去行李架看一眼。
- **北站/第10区注意事项**: 旅舍所在的第 10 区人流复杂，夜间尽量避免在偏僻的小巷独行，保持在 Rue La Fayette 等主干道走。

### 🍴 2. 法国生活“潜规则”
- **用餐时间**: 法国餐厅午餐通常在 12:00-14:00，晚餐从 19:30 开始。错过时间可能只剩快餐。
- **省钱点餐**: 餐厅的依云矿泉水很贵。请直接对侍者说：*"Une carafe d'eau, s'il vous plaît"* (请给我一瓶自来水)，这是免费且默认提供的。
- **公厕与硬币**: 欧洲公厕较少且收费(约 €1-2)。身上请时刻保留少量欧元硬币，或进入咖啡馆买杯最便宜的 Espresso 换取使用厕所的机会。

### 🆘 3. 应急预案
- **护照备份**: 请务必在云端（如百度网盘或邮件）存一份护照和签证的扫描件。
- **共享行程**: 建议将这份指南共享给国内的朋友或家人，并在社交软件（如微信）上给紧急联系人开通“共享实时位置”。
- **礼貌社交**: 在法国寻求任何帮助（问路、点餐、领钥匙）前，请务必先微笑说一句 **"Bonjour!"**。在法国文化中，没有这句问候直接提问会被视为非常无礼，而一句热情的 Bonjour 会让对方变得友好许多。
- **突发罢工**: 法国铁路偶尔会有罢工。请在每段跨城火车出发的前一天，通过 **SNCF Connect App** 再次确认车次状态（若显示 "Supprimé" 即为取消，需及时联系柜台改签）。
